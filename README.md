# Find Product Project
This program can find products from trajectory output of the Venus chemical dynamics program.

## Structure
- `find_product.py`: Entry point for the application.
- `requirements.txt`: List of dependencies.
- `tests/`: Directory for test files.

## Getting Started
2. Install dependencies:
   ```bash
   python -m pip install -r requirements.txt
   ```
3. Run the main script:
   Change the settings inside find_product.py.
   ```bash
   python find_product.py
   ```

## Testing
Add your test files in the `tests/` directory.

This project is ready for Python development in VS Code.
