# Find Product Python Project

This is a basic Python project scaffolded for development in Visual Studio Code.

## Structure
- `main.py`: Entry point for the application.
- `requirements.txt`: List of dependencies.
- `tests/`: Directory for test files.

## Getting Started
1. Activate the virtual environment (already configured).
2. Install dependencies:
   ```bash
   /home/akash/work/VSCODE/BAH_FPS/find_product_python/.venv/bin/python -m pip install -r requirements.txt
   ```
3. Run the main script:
   ```bash
   /home/akash/work/VSCODE/BAH_FPS/find_product_python/.venv/bin/python main.py
   ```

## Testing
Add your test files in the `tests/` directory.
